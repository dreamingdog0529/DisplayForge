# Roadmap

## Project Board

The project roadmap is tracked with GitHub Issues, milestones, and projects:

**[Issues](https://github.com/dreamingdog0529/DisplayForge/issues)** ·
**[Milestones](https://github.com/dreamingdog0529/DisplayForge/milestones)** ·
**[Projects](https://github.com/dreamingdog0529/DisplayForge/projects)**

These provide:
- Real-time status of planned features
- Release milestones and timelines
- Priority and progress tracking
- Links to related issues and PRs

## Themes (near term)

Ideas under consideration (not commitments):

- More reliable apply on Windows 11 edge cases
- Better multi-monitor matching when hardware changes
- DPI / HDR / advanced display property support
- UX polish for the layout editor and tray workflow
- Localization quality improvements (community corrections welcome)

## How to Contribute

### Suggesting Features

1. Check [Issues](https://github.com/dreamingdog0529/DisplayForge/issues) for existing plans
2. Open a [Feature Request](https://github.com/dreamingdog0529/DisplayForge/issues/new?template=feature_request.yml) or [Proposal](https://github.com/dreamingdog0529/DisplayForge/issues/new?template=proposal.yml)

### Discussing Roadmap

- Participate in [GitHub Discussions](https://github.com/dreamingdog0529/DisplayForge/discussions)

## Release Cadence

This project uses automated releases via Release Please (conventional commits on `main`).
See [GitHub Releases](https://github.com/dreamingdog0529/DisplayForge/releases) for the latest versions and [CONTRIBUTING.md](.github/CONTRIBUTING.md) for the process.
